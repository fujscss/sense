# SENSE壁纸

## 简介

- 刚学习Vue.js, 写了一个精选壁纸推荐的小应用练练手, 该项目使用vue-cli构建、打包, 配合vue全家桶（vue、vuex、vue-router）进行开发、使用axios进行资源请求

- Demo中api使用模拟数据, 页面布局设计使用极简风格

- 欢迎issue、fork、star😁
